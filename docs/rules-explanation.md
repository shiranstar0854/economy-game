# 第一版模拟规则

第一版使用简化规则，不用于预测真实经济。

1. 降息会提升贷款需求，加息会压低贷款需求。
2. 放贷宽松会提高短期信用扩张，但会推高坏账风险。
3. 流动性投放会抬升股票指数和风险偏好，流动性收紧会产生相反影响。
4. 信用扩张会提升企业投资和 GDP，但过热会推高通胀。
5. 坏账、失业、通胀偏离目标和市场下跌会共同推高系统性风险。

## 每轮解释方式

每一轮结果分成四个方向：

1. 企业融资变化：观察信用增速、企业债务和利润压力。
2. 居民消费变化：观察消费、收入信心和债务压力。
3. 银行风险变化：观察坏账率和资本缓冲。
4. 市场价格变化：观察股票指数、债券收益率和风险偏好。

系统总结会把这些变化压缩成一句因果解释，帮助玩家理解“政策选择如何通过资金流影响增长和风险”。

## 三条演示路径

### 连续宽松

连续选择“降息 / 宽松 / 投放”。预期表现：

- 信用扩张和 GDP 短期上升。
- 通胀和资产价格更容易上升。
- 后续坏账率和系统性风险可能累积。

### 连续收紧

连续选择“加息 / 严格 / 收紧”。预期表现：

- 通胀压力下降。
- 信用扩张放缓，企业和市场承压。
- 股票指数和风险偏好可能下降。

### 中性政策

连续选择“不变 / 正常 / 不变”。预期表现：

- 系统变化较平稳。
- 风险不会快速下降，但也不应突然进入危机。
