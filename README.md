# starshiran0854.github.io

金融系统沙盘模拟。

## 金融系统沙盘

一个回合制资金流动模拟器，用来观察利率、放贷标准和流动性如何影响经济增长、通胀、银行风险、市场价格与系统性风险。

第一版不是股票交易模拟器，也不接入真实金融数据；它的目标是帮助理解金融系统里的因果链。

## 项目边界

- 不预测真实市场。
- 不接入真实行情、真实银行数据或外部 API。
- 不模拟股票买卖、基金投资或真实交易。
- 只做教学型沙盘：用简化规则解释“政策选择 → 资金流变化 → 增长/风险变化”。

## 本地运行

```bash
npm install
npm run dev
```

开发入口是 `/starshiran0854.github.io/app.html`。仓库根目录的 `index.html` 是 GitHub Pages 使用的静态构建产物。

构建检查：

```bash
npm run build
```

## 第一版功能

- 显示当前经济状态：GDP 增长率、通胀率、失业率、银行坏账率、股票指数、系统风险。
- 玩家每轮选择 3 类决策：基准利率、银行放贷标准、流动性。
- 决策区会基于当前选择预演下一轮五部门状态。
- 点击“下一轮”后，模拟器根据简单规则更新经济状态。
- 展示本轮结果、关键指标增减值、因果解释和历史趋势图。
- 保留 10 回合限制，到第 10 回合后禁止继续推进。

## 五个沙盘部门

1. 居民部门：收入、消费、储蓄、债务、信心。
2. 企业部门：收入、债务、利润、违约风险。
3. 银行部门：存款、贷款、坏账率、资本缓冲、流动性。
4. 金融市场：股票指数、债券收益率、风险偏好、波动率。
5. 政府/央行政策：基准利率、放贷标准、流动性投放。

## 建议演示路径

- 连续宽松：多轮选择降息、宽松放贷、流动性投放，观察增长、通胀、坏账和资产价格风险。
- 连续收紧：多轮选择加息、严格放贷、流动性收紧，观察通胀回落、增长承压和市场下跌。
- 中性政策：多轮保持不变、正常放贷、流动性不变，观察系统是否相对稳定。
